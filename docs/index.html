Inicio Proyecto: LifeTracker

Aplicación web de seguimiento de vida personal

Registrá conciertos, libros, películas, ciudades y lugares — y recibí un resumen anual estilo Spotify Wrapped de todo lo que viviste.

✦ Características principales:

  • Multi-usuario con autenticación JWT
  • 5 categorías de registro (eventos, películas, libros, ciudades, lugares)
  • Resúmenes mensuales y anuales on-demand
  • Skill CLI interactivo `/add-entry`
  • Diseño temático Harry Potter — pergamino oscuro y dorado mágico

Stack Tecnológico + Superpowers

Backend

  • Python 3.11 + FastAPI
  • PostgreSQL + SQLAlchemy
  • Alembic (migraciones)
  • JWT Auth (Bearer token)

Frontend

  • Vue.js 3 + Vite
  • Pinia (estado global)
  • CSS 100% custom
  • Responsive design

Lo construido

Backend (Python/FastAPI)

  • Modelo Goal con constraint único
  • Migración Alembic reutilizando tipo PostgreSQL
  • 4 endpoints REST con auth JWT
  • Progreso calculado en query time

Frontend (Vue 3/Pinia)

  • Store goals.js con refresh automático
  • GoalCard.vue con progreso circular
  • GoalsView.vue y GoalFormView.vue
  • Widget en Dashboard con mini-ring
